### 1순위: Supabase SQL 직접 실행 (대표님 직접)
> Supabase 대시보드 → SQL Editor에서 실행

```sql
-- parking_lots org_id 채우기 (대시보드 주차장 현황 표시 필수)
UPDATE parking_lots pl
SET org_id = s.org_id
FROM stores s
WHERE pl.store_id = s.id
AND pl.org_id IS NULL;

-- 근태 공휴일 분류용 컬럼 (근무자 공휴일 집계 필수)
```
