> 타겟: 동종 발렛서비스 업체 + 중소 유료주차장 운영사 (B2B SaaS)

## 2. 페르소나

### 발렛서비스 업체
- 크루 관리를 카톡 단톡방으로 진행
- 일일 정산을 엑셀/수기로 처리
- 매장 확장 시 관리 한계
- 사고 발생 시 기록 추적 불가

### 중소 유료주차장 운영사
- 종이주차권 인쇄비 부담
- 현금 결제 비중 높아 매출 파악 어려움
- 월주차 만기 안내 수작업
- 주차장 점유율 실시간 파악 불가

---

## 3. 브랜드 포지셔닝

> **"현장 17년이 만든 전자주차권 시스템"**

### 핵심 메시지
- "주차 운영, 아직 손으로 하고 계십니까."
- "미팍티켓. 현장에서 태어난 시스템입니다."
- "6개월은 무료입니다. 직접 경험하고 판단하세요."

### 대표 스토리 (핵심 차별점)
- 2009년 주차장 주말알바 시작
- 현금정산 체계의 비효율을 직접 경험
- 매장 디지털화의 꿈 → 미팍티켓 개발
- 강남·수도권 80곳+ 직접 운영

## 4. 랜딩페이지 구조 (8섹션)

| 섹션 | 역할 | 핵심 카피 |
|------|------|-----------|
| 1. 히어로 | 3초 승부 | "아직 손으로 하고 계십니까" + "현장 17년" |
| 2. 페인포인트 | 공감 | "익숙해서 못 느끼는 비효율" |
| 3. 솔루션 | 해결책 | "현장의 문제를 기술로 풀었습니다" |
| 4. 스토리 | 차별화 (핵심) | "시작은 주차장 주말알바였습니다" |
| 5. 숫자 | 검증 | "17년의 결과입니다" (80곳, 100명, 5건, 17년) |
| 6. 가격 | 리스크 제거 | "6개월 무료, 약정·위약금 없음" |
| 7. FAQ | 이탈 방지 | 5개 질문 |
| 8. CTA | 전환 | "현장에서 태어난 시스템, 현장에서 확인하세요" |

### 카피라이팅 원칙
- 기술 용어 배제 (SaaS, PWA, Realtime 등 사용 금지)
- 단정형 문체 (물음표 최소화)
- 1~2줄 압축 (세련됨은 짧은 문장에서)
- 비용이 아닌 절감을 강조
- 모든 섹션에서 CTA 접근 가능

## 5. 채널별 마케팅 전략

### 1순위: 검색광고 + 랜딩페이지 (예산 40%)
- 키워드: "발렛 관리 프로그램", "주차장 운영 솔루션", "전자주차권", "주차장 POS"
- 랜딩페이지 CTA: "6개월 무료 도입 신청"
- 네이버 SA 일 예산 3~5만원 시작 → 성과 키워드 집중

### 2순위: 레퍼런스 콘텐츠 (예산 30%)
- 기존 운영 매장 도입 효과 수치화
- 원페이저 + 짧은 영상 제작
- 영업 미팅 + 랜딩페이지 동시 활용

### 3순위: 아웃바운드 (예산 20%)
- 네이버 지도 기반 타겟 리스트 수집
- "6개월 무료 도입 제안서" 이메일/전화

### 4순위: 콘텐츠/SEO (예산 10%)
- 네이버 블로그 실무 콘텐츠
- 장기 SEO 투자

## 6. 타임라인 (앱 출시 5월 1일 기준)

| 기간 | Phase | 핵심 활동 |
|------|-------|-----------|
| 3~4월 | 기반 구축 | 랜딩페이지 + 검색광고 + 레퍼런스 |
| 5월 1일 | 🚀 앱 출시 | 보도자료 + 프로모션 캠페인 |
| 5~6월 | 확장 | 앱 출시 효과 극대화 + 채널 확대 |
| 7~8월 | 안정화 | 유료 전환 시작 + 소개 프로그램 |

### 앱 출시 전 메시지
> "웹으로 바로 시작, 앱으로 더 강력하게"

### 앱 출시 후 메시지
> 관리자앱 + 크루앱 + 고객앱 3종 풀 라인업
